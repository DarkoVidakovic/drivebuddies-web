# SEO Launch & Search Console Checklist (M10)

Status as of the M10 hardening pass. Run `node scripts/seo-audit.mjs` after any
content/route change — it must report **0 broken links, 0 issues**.

## Sitemap & robots

- Sitemap index: `https://drivebuddies.ch/sitemap-index.xml` (→ `sitemap-0.xml`)
- `robots.txt` allows all and references the sitemap index.
- `/sitemap.xml` is rewritten (200) to the index for convenience.
- 44 indexable URLs. Excluded from the sitemap (via `astro.config.mjs` filter):
  the 404 page and the thin `/blog/tags/*` listing pages (both `noindex`).
- Every sitemap URL carries `de-CH` / `en` / `x-default` hreflang alternates,
  resolved from the central route map (`src/i18n/routes.ts`) so divergent
  DE/EN slugs pair correctly.

## Google Search Console submission

1. Verify the property `https://drivebuddies.ch` (DNS TXT record is the most
   robust method for the apex domain; Netlify DNS supports it).
2. Submit `https://drivebuddies.ch/sitemap-index.xml` under **Sitemaps**.
3. Use **URL Inspection** → *Request indexing* for the homepage (DE + EN) and
   the top money pages: `/preise/`, `/dienstleistungen/`, `/fuehrerschein/`.
4. Set the **international targeting** none-override (hreflang is handled
   on-page and in the sitemap; no manual country targeting needed).
5. After ~a week, check **Pages** for `Excluded` reasons and **Enhancements**
   for the structured-data report (BreadcrumbList, FAQ, Course, HowTo, etc.).

## 301 redirect map

Configured in `netlify.toml`:

- **Legacy Webflow blog slugs → new slugs** (3). These are the confirmed live
  old URLs from the previous site.
- **Naive cross-locale slug 301s** (28). The pre-M10 build linked to
  prefix-swapped paths (e.g. `/en/kontakt/`, `/about/`, `/services/car/`) that
  404'd because DE/EN slugs are translated. These now 301 to the canonical
  page in the correct locale.

> **Before launch:** crawl the *current production* drivebuddies.ch (Webflow)
> with Screaming Frog or `wget --spider`, export all indexed URLs, and confirm
> each maps to a destination here. Add any remaining old URLs as 301s. The map
> above covers all URLs the new site itself has ever emitted.

## hreflang matrix

- Every routed page renders reciprocal `<link rel="alternate">` for `de` / `en`
  plus `x-default` → DE (verified by `scripts/seo-audit.mjs`).
- `noindex` pages (404, blog tag listings) intentionally carry no hreflang.

## Structured data (JSON-LD)

Global `DrivingSchool`/`LocalBusiness` on every page, plus per-page:
`WebSite`+`Organization`+`FAQPage` (home), `Course`, `HowTo`, `FAQPage`,
`BreadcrumbList`, `ItemList`, `AboutPage`, `ContactPage`, `BlogPosting`,
`AggregateRating`. Validate with the
[Rich Results Test](https://search.google.com/test/rich-results) post-deploy.
